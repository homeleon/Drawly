# Drawly  (.drly)
Simple draw language to interact with LLM

Контекст: Графический движок 800x600.

Система координат: Математическая. (0, 0) — центр экрана. 
Ось X вправо, ось Y вверх.
```
(-400,300)-------------------------------(400,300)
     |                                       |
     |                                       |
     |                                       |
     |                                       |
     |                                       |
     |                 (0.0)                 |
     |                                       |
     |                                       |
     |                                       |
     |                                       |
     |                                       |
(-400,-300)------------------------------(400,-300)
```

Основной синтаксис:
Команды пишутся в формате command(arg1, arg2, ...). Пробелы игнорируются.

Команды рисования:

line(x1, y1, x2, y2) — линия.  
rect(x, y, w, h) / rect_f(x, y, w, h) — прямоугольник (центр x,y; ширина w, высота h). _f — заливка.  
circle(x, y, r)  / circle_f(x, y, r)  — круг (центр x,y; радиус r).  
oval(x, y, w, h) / oval_f(x, y, w, h) — овал (вписан в прямоугольник w на h).  
polygon(x1, y1, x2, y2, ...) / polygon_f(...) — произвольная фигура по точкам.  
text(x, y, size, "текст") — вывод текста.  

Настройки состояния:

color(name_or_hex) — задает текущий цвет (например: red, blue, #FF0055).  
width(n) — толщина линий (целое число).  
bg(color) — меняет фон всего холста.  

Система Спрайтов (Макросы):

Определение: sprite Name { commands }  
Вызов: Name(x, y, [scale])  
При вызове спрайта все координаты внутри него смещаются на (x, y) и умножаются на scale (по умолчанию 1.0).  
Спрайты могут вызывать другие спрайты.  


Файлы как спрайты:

file_sprite(x, y, scale, 'cyber_tree.drly') - вставляет картинку из файла в текущую сцену

это позволяет разбивать большую сцену на отдельные части, уменьшает нагрузку на "мозг" рисующего

Пример кода example.drly:
```
bg(#333333)

sprite tree {
  color(brown)
  rect_f(0, -20, 10, 40)
  color(green)
  polygon_f(-30, 0, 30, 0, 0, 50)
}


tree(-100, 0, 1.5)
tree(100, 50, 0.8)

color(white)
text(0, 240, 15, "Drawly Forest")
```
